# Global Compliance Auditor - Multi-Agent System

This is the repository for the **Global Compliance Auditor** multi-agent system.
For the complete project blueprint and details, please refer to [GLOBAL_COMPLIANCE_AUDITOR_PROJECT.md](../GLOBAL_COMPLIANCE_AUDITOR_PROJECT.md).

## Quick Start

1. Create python virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure environment variables in `.env` based on `.env.example`.

4. Run verify setup:
   ```bash
   python verify_setup.py
   ```
